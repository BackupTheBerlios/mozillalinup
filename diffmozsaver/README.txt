Welcome to DiffMozSaver
=======================

DiffMozSaver is a clone of the Windows Software MozBackup.
It's a utility for creating backups of Mozilla, Mozilla Firefox 
and Mozilla Thunderbird profiles and allows you to backup mail, favorites, contacts, etc. 

It's written in FreePascal (Lazarus) and currently compiles on
Linux and Windows.
